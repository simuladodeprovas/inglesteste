## 📋 About The Project

This is an interactive **English Proficiency Test** in the style of **CEBRASPE (RIGHT/WRONG)**. The test contains **30 questions** covering essential aspects of the English language, including grammar, vocabulary, reading comprehension, idioms, and pronunciation.

Perfect for students preparing for English proficiency exams, language learners, or anyone wanting to assess their English skills!

### 🎯 Topics Covered

| Category | Questions | Skills Tested |
|----------|-----------|---------------|
| **Grammar** | 1-8 | Verb tenses, conditionals, relative clauses, subject-verb agreement |
| **Vocabulary** | 9-14 | Word meanings, prefixes, common confusions |
| **Reading Comprehension** | 15-20 | Text interpretation, understanding context |
| **Idioms & Expressions** | 21-25 | Common English idioms and their meanings |
| **Pronunciation** | 26-30 | Silent letters, homophones, pronunciation differences |

## ✨ Features

- ✅ **30 questions** in CEBRASPE style (RIGHT/WRONG)
- ⏱️ **60-second timer** per question (simulates real test conditions)
- 📊 **Progress bar** and counter (1/30, 2/30...)
- 🎯 **Immediate visual feedback** (green for correct, red for incorrect)
- 🏆 **Final classification** (Beginner, Basic, Intermediate, Advanced)
- 💾 **Save your result** with name
- 🔗 **Button for full answer key** (configurable link)
- 📱 **Fully responsive** (works perfectly on mobile devices)
- 🎨 **Professional color scheme** (dark blue and yellow)
